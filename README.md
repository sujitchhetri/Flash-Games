# web-games
Flash games for you.
To play the games.

Step-1 : You need to install a flash extension from the chrome extension called flash player for the web   https://chromewebstore.google.com/detail/flash-player-for-the-web/omimccinlhlkpjaeaocglgmkbelejlhj .

Step-2 : Then go to extension and enable it. 

Step-3 : Click the game icon to open it and play.
